## Webhook relay for sending event alerts from CVP to Discord

### Usage

Run this script on any server that can communicate with CVP and has internet access to forward the events to Discord

```

python cvp_discord_alerter.py --discordURL https://discordapp.com/api/webhooks/{webhook.id}/{webhook.token}

```
